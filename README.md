## Local Setup (Developer Machine)
1. Install Flutter SDK: https://docs.flutter.dev/get-started/install
2. Create the app scaffold:
   - `flutter create --org com.kartopolo mylab_ui_flutter`
   - Move contents into this repo folder (or create directly if you prefer a different folder name).
3. Run:
   - `flutter pub get`
   - `flutter run`
